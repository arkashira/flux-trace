# Partner Targets

## Integration Roadmap

### 1. **GitHub**
- **Rationale**: GitHub is the most widely used platform for version control and collaboration. Integrating with GitHub will allow users to trace changes directly from their repositories.
- **Free-tier limits**: 5000 API requests per hour.
- **Integration effort**: S
- **Value-add**: Solves the job of tracking changes and maintaining version history seamlessly.

### 2. **Jira**
- **Rationale**: Jira is a popular project management tool used by development teams. Integrating with Jira will allow users to trace changes and issues directly from their project management dashboard.
- **Free-tier limits**: 10,000 API requests per month.
- **Integration effort**: M
- **Value-add**: Solves the job of linking changes to project tasks and issues.

### 3. **Slack**
- **Rationale**: Slack is widely used for team communication. Integrating with Slack will allow users to receive notifications and updates about changes directly in their Slack channels.
- **Free-tier limits**: 8000 messages per month.
- **Integration effort**: S
- **Value-add**: Solves the job of real-time communication and collaboration.

### 4. **Docker Hub**
- **Rationale**: Docker Hub is a popular repository for container images. Integrating with Docker Hub will allow users to trace changes in their container images.
- **Free-tier limits**: 100,000 pulls per month.
- **Integration effort**: M
- **Value-add**: Solves the job of tracking changes in container images and ensuring consistency.
